# 画面設計書 — NudgeMe

## 1. 画面一覧

| 画面ID | 画面名 | URL | 認証 | 説明 |
|--------|--------|-----|------|------|
| SCR-01 | ログイン | `/login` | 不要 | メール・パスワードでログイン |
| SCR-02 | ユーザー登録 | `/register` | 不要 | 新規アカウント作成 |
| SCR-03 | 性格診断クイズ | `/quiz` | 必要 | 初回のみ表示される性格診断（スキップ不可） |
| SCR-04 | 決断入力 | `/` | 必要 | 問題と選択肢を入力してAIに判断させる |
| SCR-05 | 決断結果 | `/decisions/[id]` | 必要 | AIの決断と理由を表示 |
| SCR-06 | 決断履歴 | `/history` | 必要 | 過去の決断一覧 |
| SCR-07 | プロフィール | `/profile` | 必要 | 性格タイプ・AIキャラクター設定 |

## 3. 各画面の概要

### SCR-01: ログイン
- メールアドレス入力
- パスワード入力
- ログインボタン
- 「アカウント登録」リンク

### SCR-02: ユーザー登録
- メールアドレス入力
- パスワード入力
- 確認パスワード入力
- 登録ボタン
- 「ログイン」リンク

### SCR-03: 性格診断クイズ
- 質問を1問ずつ表示
- 2択で回答
- 最後に結果（性格タイプ）を表示

### SCR-04: 決断入力（ホーム）
- 悩みを入力するテキストフィールド
- 選択肢を2〜4つ入力
- AIキャラクター選択（毒舌 / 優しい / 体育会系）
- 「AIに決めてもらう」ボタン
- 「ランダムに決める」ボタン

### SCR-05: 決断結果
- AIが選んだ選択肢をハイライト表示
- 決断理由のテキスト
- 「後悔した / しなかった」フィードバックボタン
- 「もう一度相談する」ボタン

### SCR-06: 決断履歴
- 過去の決断のカード一覧（質問・選択・日時・後悔フラグ）
- 各カードをクリックで詳細（SCR-05）へ

### SCR-07: プロフィール
- 現在の性格タイプ表示
- AIキャラクター変更セレクト（毒舌 / 優しい / 体育会系）
- 「性格を再診断する」ボタン（/quiz へ遷移）
